StructJumper

It can be difficult for a blind developer to understand and navigate through a large amount of code quickly, as they are unable to skim as easily as their sighted counterparts. To help blind developers overcome this problem, I created an eclipse plug-in which allows blind programmers to navigate in the code more easily.

Tactile Graphics with a Voice

In order to make graphics accessible to the blind, they are embossed on paper and the text is generally embossed using Braille. However, Braille cannot be read by a large portion of the blind community. I worked with a group to design and evaluate a system where the text on the images is turned into QR codes which can be scanned using a smartphone application.
